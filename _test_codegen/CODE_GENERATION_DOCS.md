# AI Code Generation & Auto-Save Feature

> **Automatic code extraction and saving from AI responses**

---

## Overview

AI Coder now automatically extracts and saves code blocks generated by the AI to disk, just like OpenCode and other AI coding agents. When you ask the AI to create code, it will:

1. **Extract** all code blocks from the AI's response
2. **Detect** the programming language automatically
3. **Suggest** appropriate filenames (class names, function names, etc.)
4. **Save** each code block as a separate file
5. **Display** summary of saved files

---

## Smart Filename Detection

The system uses intelligent filename suggestion:

### 1. Detects Class/Function Names

| Code | Generated Filename |
|------|-------------------|
| `class UserController` | `UserController.php` |
| `def calculate_total()` | `calculate_total.py` |
| `func main()` | `main.go` |

### 2. Detects Filename in Comments

If AI includes filename hints:
```php
// Filename: custom_controller.php
<?php
class Custom {}
```
→ Saves as `custom_controller.php`

### 3. Language-Based Defaults

| Language | Default Filename |
|----------|-----------------|
| PHP | `script.php` |
| Python | `script.py` |
| Go | `main.go` |
| JavaScript | `script.js` |
| HTML | `index.html` |
| CSS | `style.css` |
| SQL | `query.sql` |

## Configuration File

Settings stored in `ai_coder_config.json`:

```json
{
  "ai_code_gen": {
    "auto_save": true,
    "output_directory": "C:\\dev\\ai_setup_llama_cpp\\_test_codegen\\ai_output",
    "backup_on_overwrite": true,
    "supported_languages": ["php", "python", "go", "javascript", "html", "css", "sql"]
  }
}
```

### Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `auto_save` | `true` | Automatically save code blocks |
| `output_directory` | `ai_output` | Where to save files |
| `backup_on_overwrite` | `true` | Backup before overwriting |
| `supported_languages` | All | Languages to extract |

## Code Extraction Logic

### How It Works

1. **Parse AI Response**
   - Looks for markdown code blocks: ```language\ncode\n```
   - Uses regex to extract language and code

2. **Clean Code**
   - Removes leading/trailing whitespace
   - Preserves formatting and indentation

3. **Suggest Filename**
   - Checks for filename patterns in comments
   - Detects class/function names
   - Falls back to language defaults

4. **Save to Disk**
   - Creates directory if needed
   - Writes code to file
   - Handles errors gracefully

### Regex Pattern

```python
pattern = r'```(\w+)?\n(.*?)```'
```

Matches:
- Opening backticks with optional language
- Code content (including newlines)
- Closing backticks

## Integration with Chat Mode

Auto-save is integrated into chat mode (Menu 5):

```python
response = self.ask_ai(question)

if response:
    # Display response
    print(f"\nAI: {response}\n")
    
    # Auto-save code blocks if enabled
    auto_save = self.config.get('ai_code_gen', {}).get('auto_save', True)
    if auto_save:
        self.save_ai_code_blocks(response, question_text=question)
```

**Flow:**
1. User asks question
2. AI responds with code
3. Code is displayed to user
4. Code blocks are automatically extracted
5. Files are saved to output directory
6. Summary is displayed

## Troubleshooting

### Files Not Saving

**Check:**
1. Auto-save enabled (Menu 12)
2. Output directory exists and is writable
3. AI response contains code blocks

**Fix:**
```
Menu 12 → Option 1 (ensure enabled)
Menu 12 → Option 2 (set valid directory)
```

### Wrong Filenames

**AI didn't include filename hints**

**Solutions:**
1. Ask AI to include filename: "Create UserController.php that..."
2. Rename files manually after generation
3. Check output directory and rename

### Directory Not Created

**Permission error**

**Fix:**
- Run as Administrator
- Choose different directory
- Check disk space
